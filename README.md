Instructions to use the Task Manager -software:

The project is run from the TaskManagerApp\src\main\java\org\example\TaskManagerApp\Main.java file.
<img width="701" height="474" alt="Näyttökuva 2026-01-20 121821" src="https://github.com/user-attachments/assets/459d8150-91e7-4ecc-bf35-27b4bce5c455" />

The Käyttäjätunnus (Username)- and salasana (Password) -fields are where the user inserts their credentials. If the user already has an account created, they can insert their credentials and press Kirjaudu sisään (Log in).
If it's the user's first time using the app, they need to click Luo käyttäjä (Create user) which will take them to the user creation -screen shown below:

<img width="753" height="505" alt="Näyttökuva 2026-01-20 122507" src="https://github.com/user-attachments/assets/f067aaf2-c708-46ba-a17b-2e0fb1d78d8a" />

The user chooses their username and enters it into the Käyttäjätunnus (Username) -field. Then they choose their password and enter it into the Salasana (password) -box and then they re-enter it into the Salasana uudelleen (Password again) -field to confirm it.
Then they click Luo käyttäjä(Create a user) to create a new user to the database.

<img width="810" height="547" alt="Näyttökuva 2026-01-20 123839" src="https://github.com/user-attachments/assets/7284a3af-8536-4ab9-b65c-fdccef4f8ec7" />

The user can add a task by writing something into the Tehtävän nimi (Task name)- field and Tehtävän kuvaus (Description of the task) -field, and then pressing Lisää tehtävä (Add task).
The task gets added to the database and is shown on the right side of the screen. By clicking a task or it's description, the task can be edited.
By clicking Tehtävän nimi (Task name) -column or Tehtävän kuvaus (Description of the task) -column at the top of the list, the user can sort the list based on the first character.

To delete a task, the user first clicks a task on the right side of the screen and then clicks Poista tehtävä (Delete a task).
<img width="590" height="423" alt="Näyttökuva 2026-01-20 124055" src="https://github.com/user-attachments/assets/bcbeed5f-9076-4a0d-8e03-11fc7f0eb0d5" />

Then a confirmation screen appears asking, if the user really wants to delete the task.
If the user chooses Ei (No) the app takes them back to the previous main task manager screen.
If the user chooses Kyllä (Yes), the task gets deleted from the list and from the database.
